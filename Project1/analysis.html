1.요구분석
A. 기능 선택화면
업무 영역 기능 선택화면
요구사항명 A-01. 사용자 서비스
개요 기능 선택을 위한 선택 화면
상세설명 1. 기능 선택 화면에 은행 로고 출력하도록 한다.
2. 기능선택 화면에는 관리자 기능, 회원 로그인, 회원 가입, 프로그램 종료가 있다.
⦁ 관리자 기능 선택
- 1번 선택 시 관리자 로그인 화면으로 이동한다.
⦁ 고객 로그인
- 2번 선택 시 고객 로그인 화면으로 이동한다.
⦁ 회원가입
- 프로그램 이용자 대해 회원가입 기능을 제공한다.
⦁ 프로그램 종료
- 프로그램이 종료된다.
제약사항 ⦁ 제시된 번호 이외의 번호를 선택한 경우
- 1(관리자 로그인), 2(회원 로그인), 3(비회원 로그인), 4(회원가입)을 제외한 다른 번호를 입력하면 프로그램이 종료됩니다.
업무 영역 기능선택화면 > 고객 회원가입
요구사항명 A-02. 고객 회원가입
개요 기능을 사용하기 위한 회원가입 및 로그인
상세설명 1. 회원가입 시 ID, PW, 이름, 주민등록번호, 계좌번호, 연락처를 입력 받는다.
2. 회원이 되면 고유한 회원 번호를 가진다.
2. 회원가입 완료하면 로그인 화면으로 넘어간다.
3. 로그인은 ID와 PW를 입력한다.
제약사항 ⦁ 회원가입 모든 항목은 필수 입력 정보로 한다.

⦁ 이름, 생년월일, 성별, 전화번호
- 이름은 한글(2~5자)로 제한한다.
- 주민등록번호는 13자리로 제한한다. ( “-“은 제외하고 작성한다. )
- 전화번호는 OOO – OOOO – OOOO 형식으로 입력

⦁ 아이디
- 길이는 8~16자로 제한한다.
- 영문자와 숫자로만 구성되어야 한다.
- 첫 글자로 숫자를 사용 할 수 없다.
- 이미 존재하는 아이디를 입력했을 시, ‘중복된 아이디입니다.’라고 알려준다.

⦁ 비밀번호
- 영문자와 숫자로만 구성되어야 한다.

⦁ 입력이 잘못 됐을 시
- 입력 후 'n개의 항목이 회원가입 형식과 일치하지 않습니다.' 라고 알려주고, '회원 가입에 실패했습니다.' 라고 알려준다.
업무 영역 기능 선택화면 > 로그인
요구사항명 A-03. 로그인
개요 관리자 화면으로 들어가기 위한 비밀번호 입력
상세설명 1. 관리자 로그인
- 관리자 로그인이 성공하면 관리자 추가, 상품 관리, 고객 관리를 할 수 있는 화면으로 넘어간다.

2. 고객 로그인
- 고객 로그인이 성공하면 메인화면으로 넘어간다.
제약사항 ⦁ 로그인 실패 시
- 입력한 ID가 다르면 ‘일치하는 아이디가 없습니다.’라고 알려준다.
- 입력한 PW가 다르면 ‘비밀번호가 일치하지 않습니다.’ 라고 알려준다.
- ‘다시 입력하시겠습니까?’ 라고 한 번 더 물어본다.
- 둘 다 틀렸을 경우에는 ‘일치하는 아이디가 없습니다.’라고 알려준다.
